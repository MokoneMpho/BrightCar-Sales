# BrightCar-Sales
Bright Motors Car Sales project, analyses transactional vehicle sales data for Bright Motors to generate actionable business insights for the Head of Sales. The goal is to improve sales performance, optimize pricing strategies, and identify high-performing products and regions.

Business Objectives
Identify top-performing car makes and models
Analyze the relationship between price, mileage, and manufacturing year
Evaluate regional sales performance
Discover customer purchasing trends
Provide recommendations to improve profitability

Tools used
SQL (Data cleaning, transformation, aggregation)
 Databricks
Power BI / Excel (Data visualization)
ETL processes (data preparation and transformation)

Dataset Description
The dataset contains historical car sales transactions, including:

Car make and model
Selling price and cost price
Units sold
Region (city/province)

 Key Analysis Performed
Revenue analysis by car make and model
Regional sales performance analysis
Price vs mileage relationship
Time-based sales trends (monthly/yearly)
Profit margin segmentation (High, Medium, Low)

Key Insights
Certain car models consistently generate higher revenue
Sales performance varies significantly across regions
Newer vehicles tend to have higher profit margins

Project Structure
/data               → Raw and processed datasets  
/sql                → SQL queries and transformations using databricks
/dashboard          → Power BI / Excel dashboards  
/presentation       → Final business presentation using Canva

Outcome
This analysis provides data-driven recommendations to help Bright Motors:
Improve sales strategy
Optimize inventory management
Target high-performing regions
Increase overall profitability
